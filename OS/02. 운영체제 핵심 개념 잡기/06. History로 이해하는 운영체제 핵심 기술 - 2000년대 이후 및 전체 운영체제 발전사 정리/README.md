# 정리1
1. 1950년대 : 운영체제 없음 (프로그램이 시스템 자원 직접 제어)
2. 1960년대 : 배치 처리 시스템
3. 1970년대 : 시분할 시스템 / 멀티 태스킹 시스템 - UNIX OS (C언어)
4. 1980년대 : GUI, 개인용 컴퓨터
5. 1990년대 : 다양한 응용 프로그램, 인터넷 발달, 오픈소스 운동 활성화
6. 2000년대 : 오픈 소스 활성화, 가상 머신, 대용량 병렬 처리 등
